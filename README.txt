This folder contains the scripts and data necessary for the inference acceleration of dynamic graph neural networks. Each folder has the following contents:

Preproessing --> has the scripts for running preprocessing step of combining the graph snapshot and extracting the overalapping and exclusive parts.
Hepth, Flicker, Epinion, Youtube --> these folders have the scripts and graph snapshot data to measure execution cycles for the HepTh, Flicker, Epinion and Youtube dataset, respectively. Each folder has its own README files for more detailed instructions.
